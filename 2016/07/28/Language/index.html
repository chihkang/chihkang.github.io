C# Multi Language

如何在網站中使用多國語系的配置?

直覺反應就會和resource檔案有關係

以下會建議使用類別庫(dll)的方式來建立，

這樣日後若是要更新時，可以直接替換該dll，而不需要重啟網站。

類別庫僅需針對Views相關的資料夾進行一模一樣的鏡像配置即可。

Note : 預設語系 記得存取權限要改成Public (Index.resx)
其它語系的存取權限是 沒有程式碼產生 (Index.en-US.resx)

資源檔修改也是頗麻煩，建議安裝 ResXManager(Free)

這樣一次可以全部語系一起修改，內建翻譯功能，相當方便。
