Hexo-upload-to-GitHub

2016年7月9日

GutHubPages

要上傳到GitHub上，首先要 NewRepository

Repository 的命名必須和GitHub帳號一樣

建立好之後，創立一個Branch，命名為gh-pages

此時要確保有兩個Branch以上

點選選單列最右邊的Settings

往下拉一點，如果有看到

就代表這個網址已經可以成為你個人使用的網站囉~!

Upload-Hexo

使用git上傳必須要先安裝 hexo-deployer-git

上傳之前必須先設定相關參數檔案 _config.yml

上面的配置是使用SSH的方式。Generating a new SSH key
HTTPS方式如下
repo: https://github.com/userName/userName.github.io.git

設定好後執行

最後，透過瀏覽器執行

https://userName.github.io/

應該就可以看到網站已經上傳上去GitHub囉~!
